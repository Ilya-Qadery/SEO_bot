###  What it does
1. Reads any Excel/CSV (column of titles).  
2. Auto-generates 1500-word **Persian SEO articles** (HTML h2/h3) via GPT-4o-mini.  
3. Logs in to your WordPress **headlessly** and saves every article as a **draft** – ready for review when you wake up.  
4. Caches articles locally so you never pay twice for the same title.
